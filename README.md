# 合同会社GAN. 公式サイト

合同会社GAN. の公式 Web サイトの公開用ソースです。静的な HTML / CSS / JavaScript で構成され、GitHub Pages（独自ドメイン `gandot.co.jp`）で配信しています。

## 構成

| パス | 内容 |
| --- | --- |
| `index.html` | トップページ |
| `company.html` | 会社情報 |
| `support-areas.html` | 対応領域 |
| `services.html` | 事業紹介 |
| `contact.html` | お問い合わせ |
| `privacy.html` | プライバシーポリシー |
| `terms.html` | 利用規約 |
| `sitemap.html` | サイトマップ |
| `styles.css` / `script.js` | 共通スタイル・スクリプト |
| `logo*.png` / `favicon.png` | ロゴ・ファビコン |

## ローカルでの確認

任意の静的サーバーでルートを配信して確認できます。

```sh
python -m http.server 8000
# → http://localhost:8000/
```
